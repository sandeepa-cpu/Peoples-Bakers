5. PayHere Setup

  1. Register at payhere.lk → get Merchant ID & Secret
  2. Test first with sandbox: sandbox.payhere.lk (PAYHERE_SANDBOX=true)
  3. Set Notify URL in PayHere dashboard:
    https://www.peoplesbakers.lk/api/payments/payhere/notify
  4. Must be public HTTPS — localhost will not work
  5. Go live: PAYHERE_SANDBOX=false + live merchant keys

6. Gmail SMTP Setup

  1. Enable 2-Step Verification on Google account
  2. Create App Password: myaccount.google.com/apppasswords
  3. Set SMTP_HOST=smtp.gmail.com, SMTP_PORT=587, SMTP_USER, SMTP_PASS
  4. If SMTP is blank, site works but no emails are sent

9. Go-Live Checklist

  • SITE_URL set to live HTTPS URL
  • Domain purchased and DNS configured
  • SSL certificate active (HTTPS working)
  • Admin password changed from default
  • ADMIN_PATH changed to secret URL
  • SMTP configured and test email sent
  • PayHere sandbox payment tested
  • All images uploaded (images/home/, products, outlets)
  • Run: npm run verify — all checks pass
