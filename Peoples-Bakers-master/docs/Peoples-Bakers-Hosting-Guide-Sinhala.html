5. PayHere Setup

  1. payhere.lk register → Merchant ID & Secret ගන්න
  2. Test: sandbox.payhere.lk (PAYHERE_SANDBOX=true)
  3. PayHere dashboard Notify URL:
    https://www.peoplesbakers.lk/api/payments/payhere/notify
  4. Public HTTPS URL ඕන — localhost වැඩ නොකරයි
  5. Live: PAYHERE_SANDBOX=false + live keys

6. Gmail SMTP (Emails)

  1. Google account → 2-Step Verification on
  2. App Password: myaccount.google.com/apppasswords
  3. SMTP_HOST=smtp.gmail.com, PORT=587, USER, PASS set කරන්න
  4. SMTP blank නම් site run වෙනවා — emails විතරක් නැ
