# Peoples Bakers Website

Portfolio website project for Peoples Bakers with branded UI updates and basic Progressive Web App (PWA) support.

## Features
- Multi-page static site (`index`, `legacy`, `products`, `login`)
- Updated Peoples Bakers brand color system
- Mobile-responsive layouts
- PWA setup (`manifest.webmanifest` + `service-worker.js`)
- Installable app behavior in supported browsers
